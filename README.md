# Git Gud

A VSCode extension used to visualize Git history for the current repository.

## Features

### Graph Visualizer

Visualize the git graph with a highly customizable multi-branch view.

### Object Inspector

See the details of every commit, tree, and object to understand exactly what Git is tracking.

### Git Blame

See who authored each change and quickly identify the reason a change exists within the code.

### Branch Management

See the status of all branches and tags. Understand how and where pointers exist within the repository.
